# Backtests Directory

This directory contains backtesting results and reports generated by the graphwiz-trader backtesting framework.

## Directory Structure

```
backtests/
├── README.md
├── {strategy_name}_{timestamp}.html    # Individual strategy reports
├── comparison_{timestamp}.html         # Strategy comparison reports
└── backtest_results_{timestamp}.yaml  # Raw results data
```

## Report Files

### Individual Strategy Reports

Named format: `{strategy_name}_{timestamp}.html`

Example: `Momentum_20240101_120000.html`

Contains:
- Performance metrics summary
- Equity curve visualization
- Drawdown analysis
- Returns distribution
- Trade history

### Comparison Reports

Named format: `comparison_{timestamp}.html`

Example: `comparison_20240101_120000.html`

Contains:
- Side-by-side strategy comparison
- Multiple metric charts
- Performance ranking

### Results Data

Named format: `backtest_results_{timestamp}.yaml`

Machine-readable YAML format containing:
- All performance metrics
- Trade history
- Configuration parameters

## Usage

Reports are automatically generated when running backtests:

```python
from graphwiz_trader.backtesting import BacktestEngine

engine = BacktestEngine()

# Run backtest
engine.run_backtest(strategy, "BTC/USDT", start_date, end_date)

# Generate reports
engine.generate_report("Momentum")
engine.generate_comparison_report()
engine.save_results()
```

## Viewing Reports

Open HTML files in a web browser:

```bash
# Linux
xdg-open backtests/Momentum_20240101_120000.html

# macOS
open backtests/Momentum_20240101_120000.html

# Windows
start backtests/Momentum_20240101_120000.html
```

## Cleaning

To remove old backtest results:

```bash
# Remove all HTML reports
rm backtests/*.html

# Remove all YAML results
rm backtests/*.yaml

# Remove all backtest files
rm backtests/*_*  # Preserves README.md
```
